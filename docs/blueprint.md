# **App Name**: ContractFlow

## Core Features:

- Contract Creation & Lifecycle Management: Enable users to input new contract details including name, price, type, funding channel, and manage updates, such as setting settlement prices and supplementary amounts, for existing contracts.
- AI Contract Data Extraction Tool: Utilize generative AI to parse unstructured text inputs, automatically extracting and populating key contract information (e.g., name, price, type, funding channel) into the creation forms.
- Detailed Payment Tracking: Provide functionality to record and manage individual payment entries for each contract, allowing for date and amount adjustments.
- Real-time Financial Calculation & Insights: Dynamically calculate and display comprehensive financial metrics per contract, including total paid, target payable amounts, remaining balances, and final retainage, adjustable by user-selected payment ratios (e.g., 95% or 97%).
- Funding Channel Overview & Allocation: Offer a centralized dashboard to monitor financial health across different funding channels, allowing users to define total budget pools and observe real-time consumption and remaining funds.
- AI Payment Reminder Email Generator: A generative AI tool to automatically draft professional payment reminder emails tailored to specific contracts based on their current payment status and outstanding amounts due.
- Persistent Data Storage: Implement a database solution to securely store and retrieve all contract, payment, and funding information, ensuring data continuity across sessions.

## Style Guidelines:

- A light color scheme to convey professionalism and clarity.
- Primary color: A deep, professional blue (#294E99) to reflect trust and reliability.
- Background color: A very light, desaturated blue (#ECF0F3) providing a clean and subtle base.
- Accent color: A vibrant blue-green (#2EB2CB) for highlights and interactive elements, adding a touch of dynamism.
- Primary font: 'Inter' (sans-serif) for all text, ensuring excellent readability and a modern, objective feel suitable for data-heavy content.
- Utilize a clean and functional icon set, with illustrative, vector-based icons that complement the professional aesthetic and highlight AI-powered features.
- Implement a clear and responsive multi-column layout for effective data organization and intuitive navigation, inspired by modern dashboard designs, including sticky headers for key sections.
- Incorporate subtle animations for state changes, loading indicators, and form feedback to enhance user experience without being distracting.
